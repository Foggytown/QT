Школьный проект по отрисовке алгебраических фракталов.

Отрисовывает множество мандельброта.

Ответ для точки в пространстве это количество шагов за которое модуль ее как комплексного числа
привысит значение inf abs. Где следующая итерация считается по формуле z_n+1=z_n^x+c и где z_0 начальная точка.

В интерфейсе программы:

z_n^x-понятно из формулы
c_r,c_i-вещественная и комплексная части числа c соответственно
inf abs, inf try-модуль после которого завершать процесс и количество шагов после выполнения которых считать, что процесс не завершится
RGB-максимальное значение каждого из цветов(цвета в точках распределяются в зависимости от ответа пропорционально)
xs,xe,ys,ye-начало и конец рассматриваемой области пространства по x и по y соответственно. Можно ввести руками или выделить мышкой на самой картинке.
